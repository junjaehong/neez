# BBEY / Neez FrontEnd (React)

## Install
- Visual Studio 확장팩 : ES7+ React.Redux.React-Native snippets
- nldejs 사이트 : https://nodejs.org/ko/download

## 명령어
1. npm i
2. npm install lucide-react
3. npm install react-router-dom
4. npm install sockjs-client @stomp/stompjs
5. npm run dev

## 기술 스택

| 영역 | 사용 기술 |
|------|-----------|
| Frontend | React, Vite |
| State Management | Context API |
| Style | CSS |
| Build Tool | Vite |
| Language | JavaScript |
